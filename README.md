# Portfolio

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Minimalist design
- Password protection
- Responsive layout
- Project showcase

## Getting Started

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

3. Copy `.env.example` to `.env.local` and set your password:

```bash
cp .env.example .env.local
```

4. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment Variables

- `PORTFOLIO_PASSWORD` - Password for accessing the portfolio (required)

## Deployment

This project is deployed on [Vercel](https://vercel.com). The deployment is automated via GitHub integration.

## Tech Stack

- [Next.js](https://nextjs.org) - React framework
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [Vercel](https://vercel.com) - Hosting
